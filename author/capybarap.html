    はじめまして、ペリカンちゃん

    ペリカンちゃんは、どんな子だろ

    どうしてお友達になろうと思ったの？

    • Python 製なので、 pip install で入るから。
    • Python 製だから、 virtualenv で環境構築できるから。

    どんな子なの？

    • 導入してみただけだと、以前使ってた Tinkerer との違いは、あまり感じません。
    • サイトジェネレータ Pelican
